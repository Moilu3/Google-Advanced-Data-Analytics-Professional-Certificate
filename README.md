## Overview

This repository contains my coursework and projects completed as part of the **Google Advanced Data Analytics Professional Certificate** program on Coursera. The program covers advanced data analytics concepts including Python programming, exploratory data analysis (EDA), statistical analysis, and data visualization.

## Certificate Program Structure

The Google Advanced Data Analytics Professional Certificate consists of 7 courses:

| Course | Title | Status |
|--------|-------|--------|
| 1 | Foundations of Data Science | Completed |
| 2 | Get Started with Python | Completed |
| 3 | Go Beyond the Numbers: Translate Data into Insights | Completed |
| 4 | The Power of Statistics | Completed |
| 5 | Regression Analysis: Simplify Complex Data Relationships | In Progress |
| 6 | The Nuts and Bolts of Machine Learning | In Progress |
| 7 | Google Advanced Data Analytics Capstone | In Progress |

## Projects in This Repository

### Course 2: Get Started with Python
- **Automatidata Project**: Introduction to Python programming with Pandas and NumPy for data manipulation and analysis.

### Course 3: Translate Data into Insights
- **Automatidata EDA Project**: Exploratory data analysis and visualization techniques to derive meaningful insights from data.

### Course 4: The Power of Statistics
- **Structure Your Data**: Data organization and structuring techniques
- **Descriptive Statistics**: Statistical measures and data summarization
- **Hypothesis Testing**: Statistical inference and hypothesis testing methods

## Getting Started

### Prerequisites

Make sure you have Python 3.8 or higher installed on your system.

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Moilu3/Google-Advanced-Data-Analytics-Professional-Certificate.git
cd Google-Advanced-Data-Analytics-Professional-Certificate
```

2. Install the required dependencies:
```bash
pip install -r requirements.txt
```

3. Launch Jupyter Notebook:
```bash
jupyter notebook
```
